Project: Budget Analysis Dashboard

Created an interactive dashboard that transforms raw expense data into actionable financial insights, enabling data-driven decision making.

Key Features

Multi-Chart Visualization

Bar charts for department comparison, pie charts for category distribution, and line charts for trend analysis.

Interactive PivotTables

Dynamic data exploration with Slicers for one-click filtering by department, category, or time period.

Real-time Insights

Automated variance calculations and conditional formatting to highlight over-budget items.

Business Impact

  • Enhanced decision-making with immediate insights
  • Reduced manual analysis time by 75%
